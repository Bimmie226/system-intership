# Ghi chép về Dockerfile 

Dockerfile có thể hình dung như một script dùng đê build các image trong container 

Dockerfile bao gồm các câu lệnh liên tiếp nhau được thực hiện tự động trên 1 image gốc để tạo ra một image mới. Dockerfile giúp đơn giản hóa tiến trình từ lúc bắt đầu đến khi kết thúc 

Trong Dockerfile có các lệnh chính sau: 

```dockerfile
FROM 
RUN 
CMD
...
```
