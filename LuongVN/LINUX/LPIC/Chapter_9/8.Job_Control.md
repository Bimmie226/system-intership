# JOB CONTROL
- Sau khi dừng 1 tiến trình bằng `Ctrl + z`, hệ thống linux cho phép ta kill tiến trình đó hoặc khởi động lại tiến trình

## Viewing Jobs
- Sử dụng lệnh `jobs` cho phép ta xem các công việc hiện tại đang được xử lý bởi shell

  ```bash
  [luongvn@dev-server:~$]# jobs -l
  [1]  21104 Stopped                 ./test1.sh
  [2]- 21107 Stopped                 ./test1.sh
  [3]+ 21109 Stopped                 ./test1.sh
  ```

- Ta có thể kill tiến trình hoặc restart lại tiến trình đó

  ```bash
  [luongvn@dev-server:~$]# kill -9 21107
  ```

  ```bash
  [luongvn@dev-server:~$]# jobs -l
  [1]- 21104 Stopped                 ./test1.sh
  [3]+ 21109 Stopped                 ./test1.sh
  ```

## Restarting Stopped Jobs
- Ta có thể khởi động lại bất kỳ tiến trình đã dừng nào bằng lệnh `bg` kết hợp với hệ số của tiến trình đó

  ```bash
  bg 1
  ```
