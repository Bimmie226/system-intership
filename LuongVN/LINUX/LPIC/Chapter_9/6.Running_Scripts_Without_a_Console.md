# RUNNING SCRIPTS WITHOUT A CONSOLE
- Ta có thể chạy 1 script ngay cả khi đã đóng terminal đó là dùng `nohup: no hang up`
- script sẽ chạy ngay cả khi:
  - Đóng terminal
  - Đăng xuất khỏi SSH
  - Mất kết nối mạng

```bash
$ nohup ./test18.sh &
[1] 19831
$ nohup: appending output to 'nohup.out'
```
- `[1]`: job number
- `19831`: PID
- Toàn bộ output thay vì hiển thị trên màn hình → được lưu vào file: `nohup.out`
