# Triển khai Webserver trên cụm K8s 

## I. Nội dung 

- Tạo webserver chạy trên docker với k8s 
- Thử nghiệm các tính năng: 
  - scale, load balancing 
  - Update images cho deployments
  - roll back 

## II. Thực hiện 

### 2.0 Tạo file deploy apache 

```bash 
mkdir ~/project
vim ~/project/apache-deployment-service.yaml
```

Nội dung: 

```yaml
---
apiVersion: apps/v1
kind: Deployment
metadata:
  name: apache2
spec:
  replicas: 1

  selector:
    matchLabels:
      app: apache2

  template:
    metadata:
      labels:
        app: apache2

    spec:
      containers:
      - name: apache2
        image: cosy294/swarm:1.0

        ports:
        - containerPort: 80

---
apiVersion: v1
kind: Service
metadata:
  name: apache2

spec:
  type: NodePort

  selector:
    app: apache2

  ports:
  - port: 5555
    targetPort: 80
    nodePort: 30080
```

Deploy apache: 

```bash
kubectl apply -f apache-deployment-service.yaml
```
