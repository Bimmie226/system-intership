## II. Linux (Ubuntu)
Sử dụng câu lệnh sau để tải Git trên Rocky 9:

```bash
sudo apt install git-all
```

Kiểm tra lại bằng câu lệnh sau:

```bash
root@ubuntu2204:~# git --version
git version 2.34.1
root@ubuntu2204:~#
```

Nếu bạn sử dụng các phiên bản distro khác Ubuntu, bạn có thể tham khảo câu lệnh đề cài đặt git [tại đây](https://git-scm.com/install/linux)
